# 🌊 TidesMusic

**TidesMusic** is a modern, Spotify-like music player for Windows with YouTube & playlist support, radio streaming, podcasts, and smooth animated UI. Fully **ad-free** and themeable! It includes animated playback controls, themes (Dexter 🩸 | Forest 🌲 | Helldivers 2 🔫), pulse effects on playlist items, volume control slider, album art display, and a modern, responsive GUI. Enjoy uninterrupted music — 🚫 No Ads!  

## 🎵 Features
- 🔎 Tides API Support & Playlist Support  
- ✨ Animated Playback Controls  
- 🎨 Themes: Dexter 🩸 | Forest 🌲 | Helldivers 2 🔫  
- 💓 Pulse Effect on playlist items  
- 🔊 Volume Control Slider  
- 🖼 Album Art Display  
- 🖥 Modern, Responsive GUI  
- 🚫 No Ads  

## 🐍 Install Python
TidesMusic requires **Python 3.11 or 3.14**. If you don’t have Python installed, download it from [Python.org](https://www.python.org/downloads/) — recommended versions are Python 3.11.x or 3.14.x. On Windows, run the installer, check **“Add Python to PATH”**, and complete the installation. On macOS or Linux, follow the instructions on the website and ensure `python3` and `pip` are available. To verify, open a terminal (Command Prompt / PowerShell / Terminal) and run `python --version` or `python3 --version`; you should see something like `Python 3.11.6`.  

## 🛠 Installing Dependencies
Before using TidesMusic, install all required Python packages. Open PowerShell and run the following command (replace the path if different):  
`& "C:\Program Files\Python311\python.exe" -m pip install --upgrade pip PyQt6 python-vlc yt-dlp requests feedparser Pillow`  
This installs: 🖥 `PyQt6` → GUI, 🎵 `python-vlc` → Audio playback, 🔎 `yt-dlp` → YouTube & playlist support, 🌐 `requests` → Web requests, 🎙 `feedparser` → Podcast RSS feeds, 🖼 `Pillow` → Image handling.
