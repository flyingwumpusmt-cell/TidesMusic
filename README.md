# 🌊 TidesMusic

**TidesMusic** is a modern, Spotify-like music player for Windows with YouTube & playlist support, radio streaming, podcasts, and smooth animated UI. Fully **ad-free** and themeable!  

---

## 🎵 Features

- 🔎 **Tides API Support & Playlist Support**  
- ✨ **Animated Playback Controls**  
- 🎨 **Themes**: Dexter 🩸 | Forest 🌲 | Helldivers 2 🔫  
- 💓 **Pulse Effect** on playlist items  
- 🔊 **Volume Control Slider**  
- 🖼 **Album Art Display**  
- 🖥 **Modern, Responsive GUI**  
- 🚫 **No Ads**

---

## 🚫 No Ads

TidesMusic is completely **ad-free** — enjoy uninterrupted music!

## 🐍 Install Python

TidesMusic requires **Python 3.11 or 3.14**. If you don’t have Python installed yet, follow these steps:

### Step 1️⃣: Download Python

- [Official Python Downloads](https://www.python.org/downloads/)  
- Recommended: **Python 3.11.x** or **Python 3.14.x**

### Step 2️⃣: Install Python

- **Windows:**  
  1. Run the installer.  
  2. Check **“Add Python to PATH”** before clicking Install.  
  3. Complete the installation.  

- **macOS / Linux:**  
  Follow instructions on the Python website and ensure `python3` and `pip` are available in your terminal.

### Step 3️⃣: Verify Installation

Open a terminal (Command Prompt / PowerShell / Terminal) and run:

```powershell
python --version
# or
python3 --version
